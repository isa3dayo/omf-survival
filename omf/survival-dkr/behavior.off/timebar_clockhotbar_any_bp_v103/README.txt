OMFS TimeBar (Clock in Hotbar: Real + MC Time) v1.0.3
=====================================================
・ホットバーのどこか(0-8)に時計があれば「R HH:MM / M HH:MM」をアクションバーに表示（§6）。
・選択中でなくてもOK。時計をホットバーから外すと消えます。
・安定版 API (@minecraft/server 1.11.0)。

配置: ~/omf/survival-dkr/behavior/timebar_clockhold_bp/
world_behavior_packs.json の version は [1, 0, 3] に更新してください（UUIDは据え置き）。
